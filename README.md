# Hotel Booking Cancellation Prediction

This project applies supervised machine learning techniques to predict whether a hotel booking will be canceled using the Hotel Booking Demand dataset.

## Project Overview
The project includes:
- data cleaning and preprocessing
- exploratory data analysis
- feature preparation
- training and comparison of 6 classification models
- model evaluation using accuracy, precision, recall, F1-score, and ROC-AUC

## Models Used
- Logistic Regression
- Linear SVC
- K-Nearest Neighbors
- Decision Tree
- Random Forest
- Gradient Boosting

## Files
- `ML_IBM_Supervised_Classification_Project_Dalileh_documented.ipynb` — final notebook
- `ML-IBM-Supervised-Classification-Project-Dalileh-documented.pdf` — PDF report

## Dataset
Hotel Booking Demand dataset from Kaggle.

## Goal
To identify the model that best predicts booking cancellations and derive useful business insights from the analysis.
